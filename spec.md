# P4 真偽判定 仕様（ルールの正本）

> このファイルがルールの正本。実装は `src/lib/p4rules.ts`（`resultColumns(state)` ほか）に集約し、
> 全ページ（盤面 `/share/[code]`・コントローラー・結果・beta）がここを参照する。
> ルール変更時は spec.md → p4rules.ts → `npm run check:spec` の順で更新・検証する。
> 検証は `tests/p4rules.spec.ts`（末尾の「検証ケース」表と一致させる）。

## 前提

- 結果は「早側」「遅側」の2系統 ＋ 視線・ほのお・つなみ。
- **雷水の早遅** と **加速の早遅** は別の軸（独立に決める）。
- **個人ギミックは加速度（動く/動かない）のみ**。緑の枠線で表現。雷水散会は個人ギミックではなく青(雷)/赤(水)の塗り。
- **「何もしない」結果は枠線だけ**で表現：みる（視線ウソ）/ 🔥そのまま（ほのおウソ）/ 🌊そのまま（つなみホント）。動く/動かないも個人ギミックなので緑枠。

## 記憶（選ぶボタン）

### 1. GC1：ホント / ウソ（1つ選ぶ）
- ホント = 雷散会 / ウソ = 水散会（GC1の雷水の種類）
- 結果の `みない/みる` 等もこの選択で入れ替わる

#### 1-a. 雷水散会 早 / 遅（1つ選ぶ）
- 「GC1の雷水」を早側に置くか遅側に置くかを選ぶ。
  - **早を選ぶ** → GC1の雷水が ①早側、GC2の雷水が ④遅側
  - **遅を選ぶ** → GC1の雷水が ④遅側、GC2の雷水が ①早側

#### 1-b. 加速度 早 / 遅（0〜1つ選ぶ）
- GC1で付与された加速度。ホント=動く / ウソ=動かない（GC1の値に従う）。
- 早/遅で結果①/④のどちらに出すかを選ぶ。

### 2. GC2：ホント / ウソ（1つ選ぶ）
- ホント = 雷散会 / ウソ = 水散会（GC2の雷水の種類。GC1とは独立）
- 置かれる側（早/遅）は 1-a の選択で確定済み（GC1の反対側）

#### 2-a. 加速度 早 / 遅（0〜1つ選ぶ）
- GC2で付与された加速度。ホント=動く / ウソ=動かない（GC2の値に従う）。
- 早/遅で結果①/④のどちらに出すかを選ぶ。
- ※ 加速は 1-b と 2-a 合わせて **最大1個**（自分に付く加速は1回だけ）。

### 3. ほのお：ホント / ウソ（1つ選ぶ）
### 4. つなみ：ホント / ウソ（1つ選ぶ）

## 結果（左から ① 〜 ⑤）

### ① 早：雷水・加速度
- 雷水散会：1-a で「早」=GC1の雷水 / 「遅」=GC2の雷水（上）
- 動く/動かない：加速（1-b/2-a）で「早」を選んだとき（下）

### ② 早：視線
- GC1 ホント = みない / ウソ = みる

### ③ ほのお
- ホント = 離れる / ウソ = そのまま

### ④ 遅：雷水・加速度
- 雷水散会：1-a で「早」=GC2の雷水 / 「遅」=GC1の雷水（上）
- 動く/動かない：加速（1-b/2-a）で「遅」を選んだとき（下）

### ⑤ つなみ
- ホント = そのまま / ウソ = 離れる

## セルに出る個数

| セル | 出るもの | 個数 |
|---|---|---|
| ① 早 雷水・加速 | 雷水散会(上) + 動く/動かない(下) | 1〜2（雷水は必ず出る） |
| ② 早 視線 | みない/みる | 0〜1 |
| ③ ほのお | 離れる/そのまま | 0〜1 |
| ④ 遅 雷水・加速 | 雷水散会(上) + 動く/動かない(下) | 1〜2（雷水は必ず出る） |
| ⑤ つなみ | そのまま/離れる | 0〜1 |

- 雷水散会：①と④の**両方**に出る（GC1=片側、GC2=反対側）。1-a必須なので、GC1とGC2両方選べば①④とも雷水が出る。
- 動く/動かない：①④合わせて**最大1個**。

## 関連性ルール（どのボタンが何を決めるか）

### 雷水散会（①早・④遅に必ず出る）

| 1-a の選択 | ① 早側の雷水 | ④ 遅側の雷水 |
|---|---|---|
| **早** | GC1の雷水（GC1ホント=雷/ウソ=水） | GC2の雷水（GC2ホント=雷/ウソ=水） |
| **遅** | GC2の雷水（GC2ホント=雷/ウソ=水） | GC1の雷水（GC1ホント=雷/ウソ=水） |

- 種類（雷/水）：GC1の分はGC1のホント/ウソ、GC2の分はGC2のホント/ウソ（独立）。
- 置く側（早/遅）：1-a で GC1 を早にすれば GC2 は自動で遅（反対側）。

### 加速度（動く/動かない）（①④合わせて最大1個）

| 選んだボタン | 出る場所 | 値 |
|---|---|---|
| 加速GC1・早 | ① 早 | GC1ホント=動く / ウソ=動かない |
| 加速GC1・遅 | ④ 遅 | GC1ホント=動く / ウソ=動かない |
| 加速GC2・早 | ① 早 | GC2ホント=動く / ウソ=動かない |
| 加速GC2・遅 | ④ 遅 | GC2ホント=動く / ウソ=動かない |

- 4ボタンのうち押せるのは1つだけ（他を押すと移動）。0個（未選択）も可。

### 視線・ほのお・つなみ

| ボタン | 出る場所 | ホント | ウソ |
|---|---|---|---|
| GC1 | ② 早視線 | みない | みる |
| ほのお | ③ | 🔥離れる | 🔥そのまま |
| つなみ | ⑤ | 🌊そのまま | 🌊離れる |
